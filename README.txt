This is an application for visualizing trajectory datas 
such as GeoLife(http://research.microsoft.com/en-us/downloads/b16d359d-d164-469e-9fd4-daa38f2b2e13/).

Load trajectory file from right "add" button,
you can load the files whose extension is ".plt".
